Substrate

  • Bài 1 - Blockchain đầu tiên với Substrate
  • Bài 2 - Thêm Pallet vào Runtime

1 phần mềm Blockchain chạy trên 1 máy tính được gọi là 1 node , các node kết nối với nhau qua giao thức mạng ngang hàng peer-to-peer (P2P) network.Để đảm bảo tính an toàn dữ liệu trong chuỗi và quá trình xử lí chuỗi thì các node dùng 1 vài dạng đồng thuận dữ liệu trong trạng thái của dữ liệu với mỗi khối dữ liệu và thứ tự xử lý các khối.

blockchain node là gì?

Tổng quan thì 1 blockchain node sẽ chứa những thành phần sau:

  • Storage : Lưu trữ dữ liệu dạng key/value
  • Peer-to-peer networking : Giao thức kết nối mạng ngang hàng
  • Consensus capabilities : cơ chế đồng thuận
  • Khả năng xữ lí dữ liệu cho thông tin bên ngoài
  • runtime : chạy chương trình blockchain

Bởi vì việc xây dựng tất cả các thành phần như trên là 1 công việc phức tạp nên hầu hết các dự án blockchain mới sẽ fork trực tiếp từ các dự án cũ.Ví dụ Litecoin, ZCash, Namecoin, and Bitcoin Cash fork trực tiếp từ mã nguồn của Bitcoin , còn Quorum, POA Network, KodakCoin, and Musicoin fork từ Ethereum.

Tuy nhiên, những blockchain cũ không thiết kế để dễ dàng chỉnh sửa nên việc fork như vậy có nhiều hạn chế.

Substrate là gì?

Substrate là 1 framework mã nguồn mở,có tính module hóa , dễ dàng phát triển các dự á blockchain.
Substrate được thiết kế mềm dẻo,dễ phát triển ngay từ đầu.Nó cung cấp các components cốt lõi để dễ dàng xây dựng 1 blockchain node.
Để dễ bắt đầu, chúng ta dùng ngay 1 mẫu(template) do Substrate cung cấp, có sẵn 1 mạng blockchain với 1 vài tài khoản mẫu.

Bắt đầu

Công việc đầu tiên của chúng ta là compile và launch 1 blockchain node với template do Substrate cung cấp.

Cài đặt Rust

Rust là ngôn ngữ lập trình viết Substrate
Hướng dẫn cài đặt Rust http://localhost/work/rustvietnam/lap-trinh-co-ban-rust/cai-dat-rust/
Sau khi cài xong ta update bản stable

rustup default stable
rustup update

Update bản nightly và target webassembly

rustup update nightly
rustup target add wasm32-unknown-unknown --toolchain nightly

Sử dung Substrate Node Template

Substrate Node Template cung cấp cho chúng ta 1 mẫu chuẩn để bắt đầu, clone Template trên github và compile thôi

git clone -b latest --depth 1 https://github.com/substrate-developer-hub/substrate-node-template

Chuyển vào thư mục

git clone -b latest --depth 1 https://github.com/substrate-developer-hub/substrate-node-template

Compile mã nguồn

cargo build --release

Cài đặt Front-end

Front-end Template sử dụng ReactJS để tạo giao diện web cho chúng ta tương tác với blockchain node.Chúng ta hoàn toàn có thể custom Front-end Template này thành hệ thống giao diện người dùng phục vụ cho dự án riêng của mình trong tương lai.
Để cài đặt Front-end Template ta cài nodeJS và Yarn trước sau đó clone Front-end Template trên github

git clone -b latest --depth 1 https://github.com/substrate-developer-hub/substrate-front-end-template

cd substrate-front-end-template

Sau đó chạy install

yarn install

Khởi chạy Substrate node

Mở terminal và Cd chuyển tới thư mục cài Substrate Node Template
Khởi chạy node ở chế độ dev , lưu dữ liệu tạm thời

./target/release/node-template --dev --tmp

Rồi,giữ terminal này chạy nhé, ta mở 1 terminal khác để chạy front end.
Tương tự ta chuyển tới thư mục cài Front-end Template sau đó chạy bằng lệnh

yarn start

Trong trình duyệt mở địa chỉ http://localhost:8000 ta sẽ thấy giao diện

Trong template cài sẵn có 1 số account với tài khoản, mình có thể dùng để thử chuyển tiền

Chúng ta thử chuyển cho 1 account nào đó xem sao nhé , nhớ minimum là 1000000000000 nhé .

Sau đó kiểm tra Events component để xem ghi nhận giao dịch

Chúng ta dùng Control-c để dừng chương trình ở terminal nhé.
