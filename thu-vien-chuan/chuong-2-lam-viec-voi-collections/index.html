Sử dụng vector

Trong các Collections thì vector là phổ biến nhất, vector là 1 mảng có thể thay đổi kích thước(size).
Ok, mở play ground https://play.rust-lang.org/ hoặc tạo file mới và compile cũng được.

fn main() {
    let cars = vec!["Toyota", "Tesla", "Lambogini"];
    println!("cars: {:?}", cars);
    let mut fruits = Vec::new();
    fruits.push("apple");
    fruits.push("tomato");
    fruits.push("pear");
    println!("fruits: {:?}", fruits);
    let last = fruits.pop();
    if let Some(last) = last {
        println!("Removed {} from {:?}", last, fruits);
    }
}

Có 2 cách khai báo vector, 1 khai báo với số phần tử có sẵn let cars = vec!["Toyota", "Tesla", "Lambogini"]; , 2 khai báo với 1 vector rỗng và thay đổi sau let mut fruits = Vec::new();
Các bạn lưu ý đoạn này

if let Some(last) = last {
    println!("Removed {} from {:?}", last, fruits);
}

Đây là cách dùng match đơn giản, với nhiều option lựa chọn hơn thì mình sẽ dùng kiểu như vầy:

match optional {

Trong JS có hàm splice để loại bỏ/thay thế các phần tử trong mảng rất tiện, chúng ta cũng có nó trong Rust

fn main(){
    let mut stuff = vec!["1", "2", "3", "4", "5", "6"];
    println!("Original stuff: {:?}", stuff);
    let stuff_to_insert = vec!["a", "b", "c"];
    let removed_stuff: Vec<_> = stuff.splice(1..4, stuff_to_insert).collect();
    println!("Spliced stuff: {:?}", stuff);
    println!("Removed stuff: {:?}", removed_stuff);
}

Thay thế các phần tử ở stuff bằng các phần tử stuff_to_insert , vị trí thay thế 1..4, nếu ta thay vec let stuff_to_insert = Vec::new(); là vector rỗng thì sẽ là remove các phần tử 1,2,3.
Trường hợp chúng ta làm việc với vector có số lượng phần tử lớn. 

let mut large_vec: Vec<i32> = Vec::with_capacity(1_000_000);
// vector large_vec được khai báo với dung lượng đến 1 triệu phần tử
large_vec.shrink_to_fit();
//nếu ta không dùng nhiều thì chúng ta có thể gọi lệnh shrink_to_fit để co dung lượng lại tới mức cần thiết để tiết kiệm bộ nhớ

Xử lí collection bằng iterators (con chạy)

iterators là vòng lặp chạy qua tất các phần tử trong collection để xử lí nó.
Với Vector 

let names = vec!["Joe", "Miranda", "Alice"];
let mut iter = names.iter();
// ta có 1 iterator bằng hàm **iter**

Với chuỗi ta ko có hàm iter mà dùng hàm char

let mut alphabet = "ABCDEFGHIJKLMNOPQRSTUVWXYZ".chars();

let mut alphabet = "ABCDEFGHIJKLMNOPQRSTUVWXYZ".chars().enumerate();
// Hàm **enumerate** tạo iterator chi tiết với kí tự và index của kí tự, chúng ta thử vòng lặp for sẽ thấy 
for (index, letter) in alphabet {
      println!("#{}. letter in the alphabet: {}", index + 1,letter);
    }
// nếu không dùng **enumerate** thì sẽ ko lấy được số index như bên trên nhé

iterators với số 

let nums = 0..10;
// Dãy từ 0 đến 10
let nums = 0..;
// Dãy từ 0 đến vô tận
let nums: Vec<_> = (1..10).collect();
// dùng hàm **collect** để chuyển iter thành vector

Thường thì ta hay dùng vòng lặp for ... in ... để duyệt con chạy, có 1 cách khác là dùng while

fn main() {
     let mut alphabet = "ABCDEFGHIJKLMNOPQRSTUVWXYZ".chars();
     while let Some(single_char) = alphabet.next() {
        println!("{}",single_char);
     }
}

//Dùng hàm next để gọi tuần tự các phần tử trong con chạy cho đến hết

Tương tự chúng ta cũng có hàm filter để lọc phần tử

fn main() {
    let countries = [
     "U.S.A.",
        "Germany",
        "France",
        "Italy",
        "India",
        "Pakistan",
        "Burma",
   ];
   let countries_with_i: Vec<_> = countries
     .iter()
     .filter(|country| country.contains('i'))
     .collect();
   println!(
     "Countries containing the letter 'i': {:?}",
     countries_with_i
   );
}
// Lọc danh sách các quốc gia có chứa kí tự 'i'
