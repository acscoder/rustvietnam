Hàm ( Function )

Trong Rust, hàm khai báo với từ khóa fn
Tên hàm chấp nhận kí tự từ a đến z, in hoa hoặc in thường , các số nguyên từ 0..9 theo sau kí tự ( ví dụ 9function là sai vì số đứng trước) và dấu gạch nối dưới(underscore) _
Các biến đầu vào bên trong dấu ngoặc tròn () , code viết bên trong dấu ngoặc nhọn {}

fn main() {
println!("Hello, world!");
}

Hàm với giá trị trả về

fn add(a:i32,b:i32) -> i32 {
a + b
}

Code trên là ví dụ hàm cộng 2 số, chúng ta khai báo dạng dữ liệu cho giá trị trả về của hàm là i32 , lưu ý code trả về có thể có keyword return hoặc không
Nhớ không được có dấu chấm phẩy ;

Closure - hàm vô danh

Closure là 1 hàm không đặt tên , Closure dùng để gán vào 1 biến hoặc đưa vào trong 1 hàm khác
Cách khai báo Closure

fn main() {
let closure_return_val = |greeting| {
        println!("Inside Closure : {}.", greeting);
    };

closure_return_val("Hello");

 }
 ```
**fn tên_hàm** không có , (biến) thay bằng |biến| , còn lại y chang.
Nếu hàm chỉ có 1 dòng lệnh ta có thể khai báo gọn
```html
fn main() {
let closure_return_val = |greeting| println!("Inside Closure : {}.", greeting);
closure_return_val("Hello");
}

closure có thể coi như là 1 biến để truyền vào trong 1 hàm số khác

fn call_me(f: F) {
    f();
}
fn function() {
    println!("I'm a function!");
}
fn main() {
    let closure = || println!("I'm a closure!");
    call_me(closure);
    call_me(function);
}

Ví dụ trên, một closure và function được dùng như là biến đưa vào hàm call_me
Đố các bạn nhận ra 2 đứa đó khác nhau như thế nào.
Closure khai báo rồi dùng là xong,biến được giải phóng khỏi bộ nhớ,còn hàm thì lâu dài.
