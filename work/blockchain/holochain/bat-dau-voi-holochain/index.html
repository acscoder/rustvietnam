Holochain

Holochain là một framework mã nguồn mở để phát triển các microservices,kết nối ngang hàng peer-to-peer, ứng dụng hoạt động ngay trên thiết bị đầu cuối mà không cần server trung tâm. Holochain dùng để:

  • Xác thực người dùng và quản lí người dùng
  • Giữ an toàn và minh bạch cho dữ liệu và business rules
  • Quản lí phân quyền cho dữ liệu công khai và public riêng tư (private)
  • Lưu trữ và truy cập dữ liệu trên hệ cơ sở dữ liệu phân tán.
  • Tự động bảo vệ với các đe dọa bảo mật
  • Triển khai và nâng cấp ứng dụng trên thiết bị người dùng
  • Chia sẽ tài nguyên giữa các thành viên trong hệ thống

Bộ đồ nghề Holochain sẽ cung cấp:

  • Một runtime để chạy và quản lí các ứng dụng
  • Một bộ phát triển (SDK) ứng dụng viết bằng ngôn ngữ Rust.
  • Tập các công cụ để kiểm thử và phát triển ứng dụng
  • Môi trường phát triển ứng dụng

Ứng dụng trong Holochain trông như thế nào

Ứng dụng trong Holochain gọi là Happ(Holochain Application)
Để viết code cho back-end ta viết code với 1 ngôn ngữ nào đó sau đó biên dịch nó thành WebAssembly bytecode.Ở đây bộ SDK cung cấp cho chúng ta làm việc với ngôn ngữ Rust.Có thể sẽ có nhiều ngôn ngữ khác được hỗ trợ sau này.
Front-end UI của ứng dụng được viết bằng 1 ngôn ngữ nào đó nhưng thường sẽ là html / js với vuejs , reactjs framework ,Front-end gửi MessagePack gọi RPC qua WebSocket
Cả Front-end và Back-end code đều triển khai trên thiết bị cuối của người dùng, không phải trên server tập trung.
Holochain hiện tại chỉ triển khai trên Linux mà thôi,những OS khác sẽ được hỗ trợ trong tương lai.
Một Happ mẫu ứng dụng chat
https://github.com/holochain/elemental-chat/

Một Happ là 1 Cargo project, chúng ta sẽ không đi vào chi tiết mà sẽ thử triển khai Happ này chạy.
Các bước thực hiện:
1- Cài đặt Holochain
2- Biên dịch code elemental-chat này thành dạng file .webhapp
3- Cài đặt Holochain Launcher sau đó mở Holochain Launcher lên, import file .webhapp để chạy ứng dụng.
4- Để test chat giữa 2 máy ta gửi file .webhapp qua máy tính khác cài đặt Holochain Launcher và chạy ứng dụng

1- Cài đặt Holochain

Yêu cầu máy tính:

  • RAM tối thiểu 4GB(Tốt nhất 16GB trở lên)
  • CPU tối thiểu 4 Core (Tốt nhất là 6 trở lên)
  • Ổ cứng hơn 30GB
  • Kết nối Internet
  • Hệ điều hành Linux

Cài Nix Package Manager

Chạy lệnh để cài Nix Package Manager

sh <(curl -L https://nixos.org/nix/install)

Sau khi cài xong, ta tắt terminal rồi logout user và login lại , mở lại terminal chạy lệnh

. ~/.nix-profile/etc/profile.d/nix.sh

Kiểm tra version của Nix để xem cài đặt đã thành công hay chưa.

nix-shell --version

Nếu thấy thông tin version là ok

Cài đặt công cụ phát triển

Cài Cachix để biên dịch nhanh hơn

nix run -f https://cachix.org/api/v1/install cachix -c cachix use holochain-ci

Đây là tùy chọn hỗ trợ không bắt buộc
Cài Holonix

nix-shell https://holochain.love

Sau khi cài xong bạn sẽ thấy terminal đổi thành

[nix-shell:~]$

kiểm tra xem holochain đã sẵn sàng chưa

holochain --version

nếu muốn thoát khỏi holochain ta dùng lệnh exit

2- Biên dịch code elemental-chat

Chúng ta clone app này từ github

git clone https://github.com/holochain/elemental-chat.git
cd elemental-chat

Chạy lệnh để build code thành file webassembly

CARGO_TARGET_DIR=target cargo build --release --target wasm32-unknown-unknown

Kiểm tra thư mục target/wasm32-unknown-unknown/release chúng ta sẽ thấy các file .wasm như sau

chúng ta tạo thư mục tên works( đặt là gì tùy bạn) để chứa các file compile dna,happ,webhapp.
Tao file DNA , chạy lệnh

hc dna init works

Nhập tên của app là elemental-chat , ( đặt là gì tùy bạn) ta sẽ có được file config dna.yaml, ta khai báo 2 zome ( module của app ) vào file dna.yaml

Đóng gói files

hc dna pack works

Ta có được file elemental-chat.dna
Có file ta tạo file happ

hc app init works

Ta sẽ có được file config happ.yaml , đặt đường dẫn tới file dna và chạy lệnh đóng gói happ

hc app pack works


Ta có được file elemental-chat.happ
Tiếp theo chúng ta tạo file .webhapp , có thể coi .happ là back-end code của ứng dụng webhapp , chúng ta cần front-end nữa, chúng ta download UI của app

git clone https://github.com/holochain/elemental-chat-ui
cd elemental-chat-ui
yarn install
yarn run build:self-hosted

Chúng ta build code js ra sản phẩm cuối, lưu ý là phải có sẵn nodejs rồi nhé, nêu chưa thì đây https://nodejs.org/en/download/
sản phẩm build ra ở thư mục dist, vào thư mục này sẽ có

Chúng ta zip tất cả vào file ui.zip (bất kì tên nào bạn muốn) sau đó copy đưa vào thư mục works nơi chứa dna và happ, quay trở lại thư mục element-chat chạy lệnh tạo config webhapp

hc web-app init works

Config file khá dễ hiểu , gồm ui và happ

Sau đó đóng gói webhapp

hc web-app pack works

Vậy là ta có file webhapp, để chạy webhapp ta cần 1 conductor , Holochain Launcher là 1 conductor tiện dụng.

3- Cài đặt Holochain Launcher

Mở https://github.com/holochain/launcher/releases/tag/v0.3.6 kéo xuống dưới cùng download file holochain-launcher_0.3.6_amd64.AppImage, đặt permissions executable cho file và click để cài đặt.
Sau đó mở Launcher lên để chạy file elemental-chat.webhapp

Click Install new app và tìm tới file elemental-chat.webhapp của chúng ta.
