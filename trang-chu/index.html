Ngôn ngữ lập trình Rust

Trang Web dành cho những người bạn yêu thích ngôn ngữ lập trình Rust. Chúng tôi không đại diện hay có liên hệ gì với team chính thức của Rust ( rust-lang.org )

Tại sao lại là Rust

Thực thi code

Code Rust chạy cực nhanh, sử dụng bộ nhớ hiệu quả.Nó không có runtime cũng ko có garbage collector.Phát huy sức mạnh trên mọi thiết bị,nền tảng và dễ dàng tương tác với ngôn ngữ lập trình khác

Độ tin cậy

Rust có hệ thống định dạng dữ liệu đa dạng, với mô hình sở hữu biến đảm bảo an toàn bộ nhớ và an toàn khi code đa luồng, loại bỏ dùm bạn nhiều lỗi trong quá trình biên dịch

Hiệu quả

Rust có tài liệu hướng dẫn tốt, bộ biên dịch báo lỗi và hướng dẫn rõ ràng, cũng với những bộ công cụ hàng đầu trong quản lí thư viện và công cụ biên dịch. Hỗ trợ multi-editor thông minh, gồm auto-completion và type inspections,auto-formatter và hơn thế nữa.

Các lĩnh vực cần Rust

Năm 2018, cộng đồng Rust đã quyết định đưa sức mạnh và hiệu quả của lập trình Rust vào các lĩnh vực dưới đây. Trong đó, bạn có thể tìm thấy nhiều thư viện lập trình chất lượng cao và những hướng dẫn hiệu quả để có thể bắt đầu.

Cài đặt Rust

Trong trường hợp cần thử code nhanh thì ta có thể xài play ground :
https://play.rust-lang.org/

Linux

Trước hết ta update apt

sudo apt update
sudo apt upgrade

Trước khi cài Rust ,ta phải đảm bảo bộ biên dịch gcc đã được cài

sudo apt-get install build-essential -y

với CentOS/RHEL thì chạy lệnh :

sudo dnf install cmake gcc -y

Sau đó chạy lệnh cài rust

sudo apt install rustc

Kiểm tra sau cài đặt

rustc --version

Window

Với window chúng ta thường cài visualstudio trước để đảm bảo bộ biên dịch gcc https://visualstudio.microsoft.com/
Sau đó download bộ cài Rust về mà cài đặt :
https://www.rust-lang.org/tools/install

MacOs

Với MacOs ta chạy lệnh :

curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

WebAssembly

WebAssembly (gọi tắt wasm) là một loại bytecode cấp độ thấp và hiệu quả cho web.
WASM cho phép bạn sử dụng ngôn ngữ khác Javascript (như C, C++, Rust...), viết chương trình với những ngôn ngữ đó, và biên dịch trước sang WebAssembly.
