Tất cả biến trong Rust đều có 1 dạng dữ liệu cố định, Rust phải biết trước dạng dữ liệu khi chạy compile.

Số (Number)

Số nguyên có 2 loại Signed( số âm và số dương ) và UnSigned( chỉ có số dương )
Signed kí hiệu là i
bao gồm i8 i16 i32 i64 i128 và isize
con số phía sau chỉ số bit của của interger.
i8 : là số nguyên 8bit vậy i8 có 2^8 = 256 giá trị --> khoảng giá trị của i8 từ -128 ... 127
Tương tự với dạng số khác, công thức chung là -2^(n - 1) ... 2(n - 1) - 1
isize : không định giới hạn

UnSigned kí hiệu là u bao gồm u8 u16 u32 u64 u128 và usize
Tương tự, công thức chung cho giới hạn bit là 0 ... 2^n - 1
vd: u8 có giới hạn từ 0 ... 255 ( 2^8 - 1 )
usize : không định giới hạn
Đối với số nguyên thì i32 là dạng mặc định, nếu ta khai báo biến mà không khai báo dạng dữ liệu như

let x = 2;

Thì dạng dữ liệu của biến x là i32

Số thực
Số thực chỉ có 2 loại f32 và f64, số âm và dương
Nếu ta khai báo biến mà không khai báo dạng dữ liệu như

let x = 2.0;

Thì rust sẽ tự nhận dữ liệu là f64 , nếu muốn dữ liệu là f32 thì khai rõ

let x:f32 = 2.0;

Một kiểu khai báo dữ liệu khác thường của rust là khai báo ngay sau giá trị

let x = 1f32;
hoặc let x = 1_f32;

Số 1 này kiểu dữ liệu float 32 bits
Thử 1 ví dụ đơn giản sau

fn main() {
let number = 1 + 30.2;
println!({}, number);
}

kết quả là

error[E0277]: cannot add a float to an integer

Không thể cộng integer và float với nhau , vì đó là 2 kiểu dự liệu khác nhau .Chúng ta phải ép kiểu dữ liệu

fn main() {
let number = 1_f32 + 30.2;
println!({}, number);
}

Như trên mình ép số 1 về kiểu f32

Kiểu dữ liệu boolean

Chỉ có true/false , có thể khai báo bool hoặc không cần.

fn main() {
    let t = true;
    let f: bool = false;
}

Kiểu dữ liệu chuỗi (string)

Chuỗi trong Rust dễ gây nhầm lẫn vì nó có 2 loại String và &str (gọi là slice)
Đây là &str

let a = "Hello, World!";

Đây là String

let b = String::from("Hello, World!");

Hoặc khai báo 1 String rỗng, sau đó thêm vào các kí tự

fn main() {
    let mut string = String::new();
    string.push_str("Hello, ");
    string.push_str("World");
    println!("{}", string);
}

Chuyển &str sang String bằng hàm to_string()

fn main() {
    let string = "Hello, World!".to_string();
    println!("{}", string);
}

Với 1 chuỗi dài thì trong code mình dùng \ để xuống dòng trong code nhưng thực tế vẫn giữ liền mạch khi in ra

fn main() {
     let long_string = "String literals
                        can span multiple lines.
                        The linebreak and indentation here ->\
                        <- can be escaped too!";
    println!("{}", long_string);
}

Kiểu dữ liệu kí tự (char)

Char có kích thước 4 byte, là kí tự đơn thuộc mã unicode https://emojipedia.org/unicode-6.0/

fn main() {
    let c = 'z';
    let z = 'ℤ';
    let heart_eyed_cat = '😻';
}

Như ta thấy char viết bên trong dấu nháy đơn ' chứ ko nháy kép " như str , ta có thể chuyển từ str về char kí tự unicode bằng hàm chars()

let mut chars = "é".chars();
// U+0065

Ngược lại cách gọi một kí tự unicode trong chuỗi (str) như sau

fn main() {
let unicode_string = "Hi unicode \u{211D}";
println!("{}", unicode_string);
}
// 211D là mã unicode

Kiểu dữ liệu mảng(Array)

Ở Rust , array là kích thước cố định , sau khi khai báo xong ko thêm ko bớt gì được nữa
dữ liệu chấp nhận trong Array cũng là 1 loại cố định , ví dụ :

fn main() {
    let int_array: [i32; 3] = [1, 2, 3];
    for i in int_array.iter() {
        println!({}, i);
    }
}

Biến int_array có kích thước là 3 phần tử , kiểu dự liệu là i32
let int_array: [i32; 3] = [1, 2, "ba"];
hay let int_array: [i32; 3] = [1, 2, 3, 4]; là không hợp lệ

Vector
Một dạng khác của mảng(array) là vector chính là mảng có thể thay đổi kích thước , thêm , bớt được

fn main() {
    let str_vector: Vec<&str> = vec!["one"," two", "three"];
    for i in str_vector.iter() {
        println!({}, i);
    }
}

Thử thêm 1 phần tử trong vector xem

fn main() {
let mut str_vector: Vec<&str> = vec![one, two, three\];
str_vector.push(four);
for i in str_vector.iter() {
println!({}, i);
}
}

Vì vậy trong Rust ta sẽ dùng vector nhiều hơn là array thông thường

Tuples
Một dạng khác của mảng(array) nữa là Tuples , khác với mảng giá trị trong Tuples sau khi khai báo thì sẽ giữ cố định như là hằng số, ko thay đổi gì nữa,Tuples khai báo với dấu ngoặc tròn

fn main() {
let pair = (1, "One");
println!("pair is {:?}", pair);
}

Truy cập giá trị trong các dạng mảng là như nhau

fn main() {
    let a = [1, 2, 3, 4, 5];
    let first = a[0];
    let second = a[1];
}

Công thức là mảng[vị trí] trong đó vị trí đầu tiên là 0.
