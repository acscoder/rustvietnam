Modules

Modules là tập hợp nhiều hàm

pub mod module_a {
pub fn a_thing() {
println!("This is a thing");
}
pub fn a_second_thing() {
a_thing();
println!("This is another thing");
}
}
fn main() {
    module_a::a_second_thing();
}

module khai báo với keyword mod , keyword pub là publish , nếu không khai báo pub thì module trên chỉ xài bên trong file thôi, không import vào file khác được.
cách gọi hàm trong module : tên_module::tên_hàm

Bố trí module theo file

Tạo 1 project mới bằng cargo new project_name
Trong folder src tạo folder test_module, trong test_module tạo file test_fun_1.rs test_fun_2.rs và file mod.rs

├── main.rs
└── test_module
    ├── mod.rs
    ├── test_fun_1.rs
    ├── test_fun_2.rs

Mở file test_fun_1.rs thêm vào 1 hàm đơn giản nhé

pub fn test_fun(){
    println!("Func Test 1");
}

Tương tự test_fun_2.rs

pub fn test_fun(){
    println!("Func Test 2");
}

Lưu ý là thêm keyword pub ở hàm nhé để nó publish, nếu ko thì chỉ dùng được nội bên trong file thôi.

File mod.rs là file cấu hình mặc định cho module , một folder có chứa file này mới là module, ta mở mod.rs

pub mod test_func_1;
pub mod test_func_2;

pub là publish module, có thể dùng ở bất kì đâu
Trong file main.rs, ta gọi test_module

mod test_module;

sau đo chạy thử hàm trong test_module ở hàm main()

mod test_module;
fn main(){ 
  test_module::test_func_1::test_fun();
  test_module::test_func_2::test_fun();
}

Như các bạn thấy đây là cách bố trí file trong Rust , ko có import,include như ngôn ngữ khác nhưng cũng khá tiện dụng.
