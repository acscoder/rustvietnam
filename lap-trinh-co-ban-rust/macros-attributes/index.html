println! là một Macro mà chúng ta dùng thường xuyên, nhưng cho đến lúc này có lẽ chúng ta cũng chưa thể hiểu được macro là gì và nó khác gì so với hàm(function).
Định nghĩa chung thì macro là 1 chương trình nhỏ thực hiện một công việc nào đó và gọi mỗi khi cần. Dĩ nhiên là chưa rõ lắm chúng ta sẽ đi lần lượt những dạng macro và khai báo macro bằng macro_rules!

1 - Macro #[derive] , sử dụng derive để đưa các tính năng vào bên trong struct và enum
2 - Attribute-like macros , định nghĩa và sử dụng attributes
3 - Function-like macros , macro dùng như một hàm

Sự khác nhau giữa macro và hàm

Macro tạo ra những đoạn code thêm vào những đoạn code khác (metaprogramming) --> giúp chương trình ngắn gọn dễ hiểu, tái sử dụng , ví dụ với derive macro

Derive Macro

struct TestStruct{
    a:i32,
    b:String
}
fn main() {
    let test_struct = TestStruct{ a:1 ,b:"hello".to_string()};
    println!("{:?}",v);
}

Đoạn code in ra biến **test_struct** , mọi việc không thuận lơi
> error[E0277]: `TestStruct` doesn't implement `Debug`

**TestStruct** cần phải có tính năng Debug thì mới có thể in ra, bên trên **TestStruct** ta thêm 
> \#[derive(Debug)]

Thì code sẽ ổn **#[derive(Debug)]** là 1 macro derive đem tính năng Debug vào **struct TestStruct** , khi compile chương trình **\#[derive(Debug)]** sẽ sản sinh ra code để phục vụ việc Debug **TestStruct** ,chúng ta không cần phải can thiệp vào bên trong **TestStruct**

**Attribute-like macros**
> \#[route(GET, "/")]
fn index() {
// code trả về html
}

Ví dụ trên là macros kiểu Attribute dùng trong các web framework , khai báo đường dẫn "/" thì ta dùng hàm **index()** để xử lí và trả về html 

**Function-like macros**
**println!** là 1 Function-like macro , macro dùng như 1 function , các Function-like macro có dấu **!** 
> println!("{} {}","hello","world");

Khác với function , macro không làm ảnh hưởng tới quyền sở hữu biến, chúng ta thử tạo 1 macro với **macro_rules!**
```html
macro_rules! add{
 // macth like arm for macro
    ($a:expr,$b:expr)=>{
 // macro expand to this code
        {
// $a and $b will be templated using the value/variable provided to macro
            $a+$b
        }
    }
}
fn main(){
 // call to macro, $a=1 and $b=2
    println!("{}",add!(1,2) );
}

Chúng ta thêm code cho macro , trong trường hợp chỉ có 1 đầu biến đầu vào

macro_rules! add{
 // first arm match add!(1,2), add!(2,3) etc
    ($a:expr,$b:expr)=>{
        {
            $a+$b
        }
    };
// Second arm macth add!(1), add!(2) etc
    ($a:expr)=>{
        {
            $a
        }
    }
}
fn main(){
    let x = add!(1,2);
    println!("{}",  x );
    println!("{}", add!(4) );
}
