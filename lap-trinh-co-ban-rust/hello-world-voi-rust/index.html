Sau khi cài đặt chúng ta gửi lời chào Hello, World! cho phù hợp với truyền thống thôi nào
Chúng ta tạo 1 folder cho code của mình, ví dụ:

mkdir hello_world
cd hello_world

chúng ta tạo file main.rs

fn main() {
println!("Hello, world!");
}

Save file và quay lại terminal, chạy lệnh

rustc main.rs

lệnh trên sẽ build code chúng ta sang file thực thi binary, để chạy file thực thi chúng la gõ lệnh

.\main.exe

Giải thích về code, hàm main() với định nghĩa fn là hàm đầu vào chạy chương trình, tên main là cố định, chúng ta không thay đổi, trong hàm dòng lệnh gọi macro println! in ra màn hình, macro nhé, ko phải hàm, hàm sẽ là println không có dấu than ! , dĩ nhiên là sau này chúng ta sẽ nói về macro 1 cách đầy đủ.
Trong Rust, các câu lệnh sẽ kết thúc bằng chấm phẩy ; trừ giá trị trả về trong hàm là phải ko có chấm phẩy, ví dụ hàm trả về kết quả phép cộng 2 số như sau :

fn add_2_number(x:i32,y:i32)->i32{
return x+y
// hoặc x+y thôi mà ko cần chữ return cũng được
}
