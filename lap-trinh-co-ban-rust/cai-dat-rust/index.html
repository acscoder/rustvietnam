Linux

Trước hết ta update apt

sudo apt update
sudo apt upgrade

Trước khi cài Rust ,ta phải đảm bảo bộ biên dịch gcc đã được cài

sudo apt-get install build-essential -y

với CentOS/RHEL thì chạy lệnh :

sudo dnf install cmake gcc -y

Sau đó chạy lệnh cài rust

sudo apt install rustc

Kiểm tra sau cài đặt

rustc --version

Window

Với window chúng ta thường cài visualstudio trước để đảm bảo bộ biên dịch gcc https://visualstudio.microsoft.com/
Sau đó download bộ cài Rust về mà cài đặt :
https://www.rust-lang.org/tools/install

MacOs

Với MacOs ta chạy lệnh :

curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

Không cần cài đặt

Trong 1 vài trường hợp cần thử code nhanh thì ta có thể xài play ground :
https://play.rust-lang.org/
