Cấu trúc rẽ nhánh if .. else

Ai học lập trình cũng biết if .. else và dùng if .. else

if x == 4 {
    println!("x is four");
} else if x == 3 {
    println!("x is three");
} else {
    println!("x is something else");
}

Đoạn code thể hiện đầy đủ cách dùng cấu trúc rẽ nhánh if .. else rồi mình cũng ko nói gì thêm.
Vậy có gì thêm nữa với if .. else trong Rust không, dĩ nhiên là có.
if .. else để gán giá trị vào biến

fn main() {
let y = if 12 * 15 > 150 {
"Bigger"
} else {
    "Smaller"
};
println!("Biến y : {}",y);
}

Code trên dùng if .. else để gửi giá trị "Bigger" hoặc "Smaller" vào biến y tùy điều kiện.

Một điểm đặt biệt nữa là cấu trúc if let

fn main() {
let dish = ("Ham", "Eggs");
if let ("Bacon", b) = dish {
    println!("Bacon is served with {}", b);
} else {
    // This block is evaluated instead.
    println!("No bacon will be served");
}
if let ("Ham", b) = dish {
    println!("Ham is served with {}", b);
} else{
    println!("No Ham will be served");
}
}

Đây là cấu trúc match pattern - so sánh theo mẫu , biến dish là ("Ham", "Eggs") so sánh với ("Bacon" khác , b) b đại diện cho 1 biến là giá trị bất kì vậy so sánh "Bacon" khác "Ham" nên cấu trúc đầu tiên là false nên in ra "No bacon will be served"
Tương tự cấu trúc thứ 2 so sánh ("Ham", b) và ("Ham", "Eggs") b đại diện cho 1 biến là giá trị bất kì vậy "Ham" bằng "Ham" và biến b sẽ gán bằng "Eggs" cho nên kết quả là true và in ra "Ham is served with Eggs"
Chúng ta sẽ học về match cũng so sánh tương tự như if let này nhưng nhiều option lựa chọn hơn

Code trong loop lặp lại liên tục cho đến khi có lệnh dừng , tương tự là với nó là while true

while true{

}

Vòng lặp For

Vòng lặp For hiện diện ở mọi ngôn ngữ lập trình , trong Rust ta dùng For .. in

let v = &["apples", "cake", "coffee"];
for text in v {
    println!("I like {}.", text);
}

Lặp qua từng phần tử trong mảng, nếu chúng ta cần index của phần tử

fn main() {
let v = &["apples", "cake", "coffee"];
for (ind,text) in v.iter().enumerate() {
    println!("I like {} {}.", ind ,text);
}
}

Một cách tương tự là dung for_each để duyệt từng phần tử trong con chạy

fn main() {
let v = &["apples", "cake", "coffee"];
v.iter().enumerate().for_each(|(i, x)| {
    println!("I like {} {}.", i, x);
});
}

Kết quả như code for .. in bên trên nhé.

Vòng lặp while

while điều_kiện {
//code here
}

Chạy liên tục trong khi điều kiện vẫn còn đúng

let mut i = 0;
while i < 10 {
    println!("hello");
    i = i + 1;
}

in ra hello 10 lần
Giống như if let ta có while let

#![allow(unused)]
fn main() {
let mut vals = vec![2, 3, 1, 2, 2];
while let Some(v @ 1) | Some(v @ 2) = vals.pop() {
    println!("{}", v);
}
}

Vòng lặp so sánh với vals.pop() hàm lấy ra phần tử cuối của vector vals so sánh với 2 hoặc ( | ) 1 nếu hợp lệ gán vào biến v để chạy code trong vòng lặp, hàm pop lấy từ dưới lên nên ta đếm 2 , 2,1 tiếp theo là 3 ko hợp lệ nên while let kết thúc

Thường với loop thì mình hay ngại while vì lỡ tay nó dẫn tới vòng lặp vô hạn ngoài ý muốn, nhưng Rust nó kiểm tra khi compile rất kĩ nên cứ yên tâm mà code nhé!!

        if i > 2{
