struct PointU32 
{
    x: u32,
    y: u32
}
struct PointF32
{
    x: f32,
    y: f32
}
struct PointI32
{
    x: i32,
    y: i32
}

Chúng ta có 3 struct gần như nhau, khác dạng dữ liệu là u32,f32,i32
Định nghĩa vậy quá rườm rà, ta dùng generic rút ngắn thành 1 struct duy nhất

struct Point<T>
{
    x: T,
    y: T
}

\ là 1 generic đại diện cho một dạng dữ liệu bất kì
Với hàm cũng vậy, ví dụ ta có 3 hàm nhận biến đầu vào là u32,f32,i32

fn foo_u32(x: u32) -> u32
{
    x*x
}
fn foo_f32(x: f32) -> f32
{
    x*x
}
fn foo_i32(x: i32) -> i32
{
    x*x
}

Việc này thật sự rườm rà, dư thừa code , ta dùng generic viết thành 1 hàm

fn foo<T>(x: T) -> T
where T: std::ops::Mul<Output=T> + Copy
{
    x*x
}

Đây là giới thiệu về generic,dĩ nhiên generic vẫn còn nhiều thứ rườm rà hay ho khác chờ ta ở các phần nâng cao hơn.
